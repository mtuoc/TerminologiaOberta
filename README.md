# TerminologiaOberta
Scripts and programs to deal with Termcat's Terminologia Oberta XML files

To download Terminologia Oberta termbases go to: [https://www.termcat.cat/ca/terminologia-oberta](https://www.termcat.cat/ca/terminologia-oberta). 

Download the XML file of the desired termbase and convert it to tab txt or to TBX.

* To convert to tab txt using the terminal, use TO2tabtxt.py (-h option shows the help).
* To convert to tab txt using a GUI interface, use TO2tabtxt-GUI.py.
* To convert to TBX using the terminal, use TO2TBX.py (-h option shows the help).
* To convert to TBX using a GUI interface, use TO2TBX-GUI.py.

  In the releases Windows executable files of the GUI programs are available.
